A basic Python program to manage your daily tasks.

## Description

This is a simple command-line to-do list application that allows users to add tasks and view all their tasks. Built as part of SEN 201 coursework.

## Features

- ✅ Add new tasks
- ✅ View all tasks in a numbered list
- ✅ Simple and easy to use

## Requirements

- Python 3.x

## How to Run

1. Make sure Python is installed on your computer
2. Download the `todo.py` file
3. Open your terminal/command prompt
4. Navigate to the folder with the file
5. Run this command:
```
python todo.py
```

## Example Output
```
Added: Buy groceries
Added: Study for SEN 201
Added: Exercise

=== MY TO-DO LIST ===
1. Buy groceries
2. Study for SEN 201
3. Exercise
```
